fragmented city PDFs and contractor blogs.

Below is a clear concept + structure you could actually build and scale.
1. Core Idea (What Makes It Valuable)
An all-in-one ADU platform for every U.S. state that answers:
“Can I build an ADU on my property, how do I do it legally, and who can help me?”
Our competitive edge would be:
Verified laws & codes
Location-specific rules
Practical building guidance
Vetted professionals
Real cost & feasibility clarity
2. Main Sections of the Website
A. State-by-State ADU Law Database ⭐ (Your backbone)
Each state page would include:
ADU legality (Allowed / Restricted / Prohibited)
Owner-occupancy rules
Max size, height, setbacks
Parking requirements
Utility connection rules
Fire & life safety requirements
Recent law updates (timeline)
Each city or county sub-page:
Zoning overlays
Coastal, historic, HOA overlays
Local amendments to state law
Permit processing time
Example URL structure:
/california/san-diego/adu-rules
B. Code & Construction Standards (Plain English)
Translate building codes into simple human language:
Detached vs attached vs Conversion ADUs
Structural requirements
Fire separation & egress
Energy code compliance
Accessibility rules (when required)
You can reference:
International Code Council
IBC, IRC, CBC, CFC summaries (no copying—interpretation only)
C.
“Can I Build?” Property Checker 🔍 (High Value Tool)
User enters:
Address
Lot size
Zoning
Existing structures
Output:
ADU allowed? (Yes / Conditional / No)
Max ADU size
Likely constraints
Estimated approval difficulty
Rough cost range
This alone could justify subscriptions later.

A guided roadmap:
Feasibility check
Survey & site constraints
Design options (garage, detached, conversion)
Plan submittal
Permit review
Construction
Final inspection & occupancy
Include:
Checklists
Typical timelines
Common rejection reasons
E. Plans, Designers & Contractors Directory
Curated listings by location:
ADU-experienced architects
Designers
Engineers
Builders
Prefab ADU companies
Filters:
State
City

F. ADU Cost Library 💰
Realistic numbers:
Design fees
Permit costs
Construction by type
Utility upgrades

Break down by:
State
Urban vs suburban
Detached vs conversion
G. Law Change Tracker & Alerts
Users can:
Subscribe to a city/state
Get notified when ADU laws change
See “before vs after” summaries
This is huge for:
Homeowners

Investors
